# COLORS Lab - LAMP Stack Application

## Description
This is a web application designed to manage a color contact list. It allows users to register, log in, add colors, and search through their existing records. It demonstrates full-stack development using the LAMP stack.

## Technologies Used
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** Apache (Linux/Ubuntu)

## Installation & Setup
1.  Clone the repository.
2.  Import the database schema (not included in repo for security).
3.  Navigate to the `api` folder and configure the database connection:
    * Open `Login.php`, `AddColor.php`, and `SearchColors.php`.
    * Replace `"See_README_for_Auth"` with your actual database password.
4.  Move the public files to your web server root (e.g., `/var/www/html`).
